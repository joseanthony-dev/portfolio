Building a photobooth and its rental website

March 2026 • Experience report — Personal project

HTML/CSSJavaScriptCloudflare

The idea

It all started with a simple observation: renting a photobooth for an event is expensive, and the available options are not always up to standard. I decided to build my own from scratch — the hardware first, then the website to offer it for rent.

Building the photobooth

Before writing any code, I had to design and assemble the hardware. The enclosure, lighting, screen, camera, printer — each component was chosen and tested to achieve a professional result. This project is as much technical (wiring, integration) as it is aesthetic (final look, ergonomics).

The website

Goal

Create a professional showcase website to present the photobooth, display rental packages and enable online booking. The site needed to be fast, responsive and inspire trust.

Features

  • Packages page: clear presentation of offers with prices and inclusions
  • Online booking: form with date selection, package choice and contact details
  • Automatic email contact: each request automatically sends an email with the details
  • Responsive design: adapted for mobile, tablet and desktop

Hosting

The site is deployed on Cloudflare with custom DNS configuration. Cloudflare provides speed (global CDN), security (automatic HTTPS, DDoS protection) and management simplicity.

Challenges

The hardware

Making all components coexist in a compact and aesthetic enclosure was not simple. Each element has its constraints: heat, wiring, accessibility for maintenance. Several prototypes were needed.

Website credibility

For a rental service, the website must inspire trust from the first visit. Design, photos, clear packages and a functional form are essential. A sloppy site means a lost customer.

Email delivery

Integrating automatic email on a static site requires a third-party service. The setup had to be reliable to never miss a booking request.

Key takeaways

  • A complete project: from hardware to web, it's rare to touch so many domains in a single project.
  • Design matters: for a commercial site, appearance and information clarity make the difference.
  • Cloudflare simplifies everything: DNS, HTTPS, CDN — in a few clicks, the site is fast and secure.
  • Test with real users: feedback from early visitors helped improve the booking flow.
